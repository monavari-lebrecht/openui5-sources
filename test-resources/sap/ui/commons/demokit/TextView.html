Introduction

The TextView control is a simple control displaying pure text. If the configured text value contains HTML or script tags, those will be escaped. Line breaks (\r\n, \n\r, \r, \n) get replaced by a <BR /> tag.

A TextView can be used to display static text, or to display domain data relevant for the execution of the application. In the second case, it might be necessary to mark the data as valid or invalid or erroneous. Using the semanticColor property, such a tagging is possible.

Using the design property, different text designs can be used, for example bold or italic.
