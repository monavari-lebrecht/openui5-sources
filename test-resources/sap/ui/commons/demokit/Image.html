Image

Introduction

The Image control is a wrapper around the HTML tag for graphics. There are two main scenarios how the Image control is used: As a visually appealing clickable area in the form of an image button, or as a purely decorative element. In the second case, images are not pre-defined as tab stops in accessibility mode. Whether or not an image is decorative can be configured with the property of the same name.
