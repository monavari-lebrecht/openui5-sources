Overview

This Demo Kit provides a comprehensive overview of the SAPUI5 framework which is a platform for developing rich user interfaces for modern Web business applications.

Content

For navigation through the Demo Kit, the following main entry points are available (see above):

  • Developer Guide (a summary of valuable information around the used programming languages, open source technologies, development tools and APIs)
  • Controls (running demo examples with descriptions and source codes)
  • API Reference (JavaScript documentation of Framework and Control API)

What's New

Development news can be found in the release notes on a regular basis.

Target Groups

Target groups are customers and SAP developers with Web development skills aiming for developing Web business applications and SAPUI5 controls.
