Method Detail
attachHandler(fnFunction, oListener?, sName)
Registers the given handler to the range change event, which is fired when a new range of the set is entered.

The handler has one map parameter mParams:

  • mParams.from: the range start value
  • mParams.to: the range end value, not defined for the last range (INFINITY)
  • mParams.unit: the used unit, e.g. px
  • mParams.name: the range name if defined
Parameters:
{Function} fnFunction The function to call, when the range change event occurs.
{Object} oListener? The 'this' context of the handler function.
{String} sName The name of the range set to listen to.

initRangeSet(sName, aRangeBorders, sUnit?, aRangeNames?, bSuppressClasses?)
Initializes a Screen width media query range set.

This function can either be called only with the name parameter to initialize a predefined range set, e.g. sap.ui.Device.media.initRangeSet(sap.ui.Device.media.RANGESETS.SAP_3STEPS).

Or it is possible to define a custom range set as in the following example: sap.ui.Device.media.initRangeSet("MyRangeSet", [200, 400], "px", ["Small", "Medium", "Large"]) defines 3 ranges:

  • 0px-199.999px with name "Small"
  • 200px-399.999px with name "Medium"
  • 400px-INFINITY with name "Large"

The range names are optional. If they are specified also a CSS class (e.g. sapUiMedia-MyRangeSet-Small) is added to the document root depending on the current active range. This can be suppressed via parameter bSuppressClasses.

Parameters:
{String} sName The name of the range set. Either a predefined or custom one. The name must be a valid id (consist of letters and digits).
{int[]} aRangeBorders The range borders
{String} sUnit? The unit which should be used. Allowed values are px (default), em or rem.
{String[]} aRangeNames? The names of the ranges. The names must be a valid id (consist of letters and digits).
{boolean} bSuppressClasses? Whether writing CSS classes to the document root should be suppressed
removeRangeSet(sName)
Removes a previously initialized range set and detaches all registered handlers.

Initialized predefined range sets (@see sap.ui.Device.media#RANGESETS) cannot be removed.

Parameters:
{String} sName The name of the range set.
