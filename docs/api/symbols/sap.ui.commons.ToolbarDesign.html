Static Class sap.ui.commons.ToolbarDesign

Determines the visual design of a Toolbar.

Field Summary
sap.ui.commons.ToolbarDesign.Flat The included controls have a very light appearance.
sap.ui.commons.ToolbarDesign.Standard The toolbar elements such as buttons for example have their normal visual design, and the toolbar appears solid.
sap.ui.commons.ToolbarDesign.Transparent The controls included in the toolbar have a normal visual design where the toolbar appears transparent.
Field Detail
