Class sap.m.ListBase

extends Control
known direct subclasses: List, Table

sap.m.ListBase Control provides a basic functionality of List and Table. Selection, deletion, unread states and inset style are also maintained here.

Note: The ListBase including all contained items is completely re-rendered when the data of a bound model is changed. Due to the limited hardware resources of mobile devices this can lead to longer delays for lists with many items. As such the usage of a list is not recommended for these use cases.


Since: 1.16.

new sap.m.ListBase(sId?, mSettings?)
Constructor for a new ListBase.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties
    • inset : boolean (default: false)
    • visible : boolean (default: true)
    • headerText : string
    • headerDesign : sap.m.ListHeaderDesign (default: sap.m.ListHeaderDesign.Standard)
    • footerText : string
    • mode : sap.m.ListMode (default: sap.m.ListMode.None)
    • width : sap.ui.core.CSSSize (default: '100%')
    • includeItemInSelection : boolean (default: false)
    • showUnread : boolean (default: false)
    • noDataText : string
    • showNoData : boolean (default: true)
    • enableBusyIndicator : boolean (default: true)
    • modeAnimationOn : boolean (default: true)
    • showSeparators : sap.m.ListSeparators (default: sap.m.ListSeparators.All)
    • swipeDirection : sap.m.SwipeDirection (default: sap.m.SwipeDirection.Both)
    • growing : boolean (default: false)
    • growingThreshold : int (default: 20)
    • growingTriggerText : string
    • growingScrollToLoad : boolean (default: false)
    • rememberSelections : boolean (default: true)
  • Aggregations
    • items (default aggregation) : sap.m.ListItemBase[]
    • swipeContent : sap.ui.core.Control
    • headerToolbar : sap.m.Toolbar
    • infoToolbar : sap.m.Toolbar
