Class sap.m.FlexBox

extends Control
known direct subclasses: HBox, VBox

The FlexBox control builds the container for a flexible box layout.

Browser support: This control is not supported in Internet Explorer 9!

new sap.m.FlexBox(sId?, mSettings?)
Constructor for a new FlexBox.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties
    • visible : boolean (default: true)
    • height : sap.ui.core.CSSSize (default: '')
    • width : sap.ui.core.CSSSize (default: '')
    • displayInline : boolean (default: false)
    • direction : sap.m.FlexDirection (default: sap.m.FlexDirection.Row)
    • fitContainer : boolean (default: false)
    • renderType : sap.m.FlexRendertype (default: sap.m.FlexRendertype.Div)
    • justifyContent : sap.m.FlexJustifyContent (default: sap.m.FlexJustifyContent.Start)
    • alignItems : sap.m.FlexAlignItems (default: sap.m.FlexAlignItems.Stretch)
  • Aggregations
    • items (default aggregation) : sap.ui.core.Control[]
  • Associations
    • Events
