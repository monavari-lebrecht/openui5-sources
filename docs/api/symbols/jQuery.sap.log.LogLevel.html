Namespace jQuery.sap.log.LogLevel

Enumeration of levels that can be used in a call to jQuery.sap.log.Logger#setLevel(iLevel, sComponent).

Deprecated API:Since 1.1.2. To streamline the Logging API a bit, the separation between Level and LogLevel has been given up. Use the (enriched) enumeration {@link jQuery.sap.log.Level} instead.
