Class sap.ui.commons.TextView

extends Control
implements ToolbarItem

Is used to display some continous text. The control can inherit the text direction from its parent control.

Constructor Detail
new sap.ui.commons.TextView(sId?, mSettings?)
Constructor for a new TextView.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties
    • text : string (default: '')
    • textDirection : sap.ui.core.TextDirection (default: sap.ui.core.TextDirection.Inherit)
    • visible : boolean (default: true)
    • enabled : boolean (default: true)
    • helpId : string (default: '')
    • accessibleRole : sap.ui.core.AccessibleRole (default: sap.ui.core.AccessibleRole.Document)
    • design : sap.ui.commons.TextViewDesign (default: sap.ui.commons.TextViewDesign.Standard)
    • wrapping : boolean (default: true)
    • semanticColor : sap.ui.commons.TextViewColor (default: sap.ui.commons.TextViewColor.Default)
    • textAlign : sap.ui.core.TextAlign (default: sap.ui.core.TextAlign.Begin)
    • width : sap.ui.core.CSSSize
  • Aggregations
    • Associations
      • ariaDescribedBy : string | sap.ui.core.Control
      • ariaLabelledBy : string | sap.ui.core.Control
    • Events
      Parameters:
      {string} sId? id for the new control, generated automatically if no id is given
      {object} mSettings? initial settings for the new control
