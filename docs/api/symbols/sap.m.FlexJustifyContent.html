Static Class sap.m.FlexJustifyContent

Available options for the layout of elements along the main axis of the flexbox layout.

Field Summary
sap.m.FlexJustifyContent.Center Box items are packed toward the center of the line.
sap.m.FlexJustifyContent.End Box items are packed toward the end of the line.
sap.m.FlexJustifyContent.Inherit Inherits the value from its parent.
sap.m.FlexJustifyContent.SpaceAround Box items are evenly distributed in the line, with half-size spaces on either end.
sap.m.FlexJustifyContent.SpaceBetween Box items are evenly distributed in the line.
sap.m.FlexJustifyContent.Start Box items are packed toward the start of the line.
