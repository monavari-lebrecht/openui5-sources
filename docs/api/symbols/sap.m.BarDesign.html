Static Class sap.m.BarDesign

Types of the Bar design


Since: 1.20.
Field Summary
sap.m.BarDesign.Auto The Bar can be inserted into other controls and if the design is "Auto" then it inherits the design from parent control.
sap.m.BarDesign.Footer The bar will be styled like a footer of the page.
sap.m.BarDesign.Header The bar will be styled like a header of the page.
sap.m.BarDesign.SubHeader The bar will be styled like a subheader of the page.
