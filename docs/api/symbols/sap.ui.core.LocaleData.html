getFirstDayOfWeek() : int
Returns the day that usually is regarded as the first day of a week in the current locale. Days are encoded as integer where sunday=0, monday=1 etc.

All week data information in the CLDR is provides for territories (countries). If the locale of this LocaleData doesn't contain country information (e.g. if it contains only a language), then the "likelySubtag" information of the CLDR is taken into account to guess the "most likely" territory for the locale.

Returns:
{int} first day of week
getIntervalPattern(sId) : string
Returns the interval format with the given Id (see CLDR documentation for valid Ids) or the fallback format if no interval format with that Id is known.
