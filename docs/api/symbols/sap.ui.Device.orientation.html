Namespace sap.ui.Device.orientation

Orientation Change Event API.

Method Summary
attachHandler(fnFunction, oListener?) Registers the given handler to the orientation change event.
detachHandler(fnFunction, oListener?) Deregisters a previously registered handler from the orientation change event.
Method Detail
attachHandler(fnFunction, oListener?)
Registers the given handler to the orientation change event.

The handler has one map parameter mParams:

  • mParams.landscape: whether the orientation is currently landscape
Parameters:
{Function} fnFunction The function to call, when the orientation change event occurs.
{Object} oListener? The 'this' context of the handler function.
detachHandler(fnFunction, oListener?)
Deregisters a previously registered handler from the orientation change event.
Parameters:
{Function} fnFunction The function to call, when the orientation change event occurs.
{Object} oListener? The 'this' context of the handler function.
