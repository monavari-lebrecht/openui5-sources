Static Class sap.m.FlexDirection

Available directions for flex layouts.

Field Summary
sap.m.FlexDirection.Column Elements are layed out along the direction of the block axis (usually top to bottom).
sap.m.FlexDirection.ColumnReverse Elements are layed out along the reverse direction of the block axis (usually bottom to top).
sap.m.FlexDirection.Inherit Inherits the value from its parent.
sap.m.FlexDirection.Row Elements are layed out along the direction of the inline axis (text direction).
sap.m.FlexDirection.RowReverse Elements are layed out along the reverse direction of the inline axis (against the text direction).
