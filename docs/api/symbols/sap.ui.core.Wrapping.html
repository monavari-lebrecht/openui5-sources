Static Class sap.ui.core.Wrapping

Configuration options for text wrapping.

Field Summary
sap.ui.core.Wrapping.Hard Inserts actual line breaks in the text at the wrap point.
sap.ui.core.Wrapping.None The standard browser behavior is considered for wrapping.
sap.ui.core.Wrapping.Off Wrapping shall not be allowed.
sap.ui.core.Wrapping.Soft The text is actually on the same line but displayed within several lines.
