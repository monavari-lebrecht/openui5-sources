Class sap.ui.commons.Panel

extends Control
known direct subclasses: Tab

Represents a container with scroll functionality, the control can be used for text and controls. The Panel does not layout the embedded controls.

Constructor Detail
new sap.ui.commons.Panel(sId?, mSettings?)
Constructor for a new Panel.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties
    • width : sap.ui.core.CSSSize (default: '100%')
    • height : sap.ui.core.CSSSize
    • enabled : boolean (default: true)
    • visible : boolean (default: true)
    • scrollLeft : int (default: 0)
    • scrollTop : int (default: 0)
    • applyContentPadding : boolean (default: true)
    • collapsed : boolean (default: false)
    • areaDesign : sap.ui.commons.enums.AreaDesign (default: sap.ui.commons.enums.AreaDesign.Fill)
    • borderDesign : sap.ui.commons.enums.BorderDesign (default: sap.ui.commons.enums.BorderDesign.Box)
    • showCollapseIcon : boolean (default: true)
    • text : string
  • Aggregations
    • content (default aggregation) : sap.ui.core.Control[]
    • title : sap.ui.core.Title
    • buttons : sap.ui.commons.Button[]
  • Associations
    • Events
