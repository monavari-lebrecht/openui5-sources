Static Class sap.m.ListMode

Different modes for the list selection (predefined modes)

Field Summary
sap.m.ListMode.Delete delete mode (only one list item can be deleted)
sap.m.ListMode.MultiSelect multi selection mode (whole list item including checkbox will be selected)
sap.m.ListMode.None default mode (no selection)
sap.m.ListMode.SingleSelect right positioned single selection mode (only one list item can be selected)
sap.m.ListMode.SingleSelectLeft left positioned single selection mode (only one list item can be selected)
sap.m.ListMode.SingleSelectMaster Single selection master mode (only one list item can be selected), selected item is highlighted but no radiobutton is visible.
