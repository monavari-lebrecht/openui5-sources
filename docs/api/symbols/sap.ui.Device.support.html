Namespace sap.ui.Device.support

Contains information about detected capabilities of the used browser or device.

Field Summary
matchmedia Flag indicating whether media queries via JavaScript are supported.
matchmedialistener Flag indicating whether events on JavaScript media queries are supported.
orientation Flag indicating whether the native orientationchange event is supported.
retina Flag indicating whether the device has a Retina display.
touch Flag indicating whether touch events are supported.
websocket Flag indicating whether WebSockets are supported.
