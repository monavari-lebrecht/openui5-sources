Class sap.ui.commons.RatingIndicator

extends Control

Is used to let the user do some rating on a certain topic. The amount of rating symbols can be specified, as well as the URIs to the image icons which shall be used as rating symbols. When the user performs a rating, an event is fired.

Constructor Detail
new sap.ui.commons.RatingIndicator(sId?, mSettings?)
Constructor for a new RatingIndicator.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties
    • visible : boolean (default: true)
    • editable : boolean (default: true)
    • maxValue : int (default: 5)
    • value : float (default: 0)
    • averageValue : float (default: 0)
    • iconSelected : sap.ui.core.URI
    • iconUnselected : sap.ui.core.URI
    • iconHovered : sap.ui.core.URI
    • visualMode : sap.ui.commons.RatingIndicatorVisualMode (default: sap.ui.commons.RatingIndicatorVisualMode.Half)
  • Aggregations
    • Associations
      • ariaDescribedBy : string | sap.ui.core.Control
      • ariaLabelledBy : string | sap.ui.core.Control
    • Events
      • change : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    Parameters:
    {string} sId? id for the new control, generated automatically if no id is given
    {object} mSettings? initial settings for the new control
