Configure KPI Tiles

The Configure KPI Tiles app allows you to create tile visualizations for an active evaluation.

The app also lets you configure the drill-down navigation path for the tile. This determines whether you navigate from the tile to the SAP Smart Business generic drill-down application or to a custom application when you click the tile at runtime.

Key Features
  • View the configured tiles for an active evaluation.

    In the Configured Tiles view, you can display the tiles that you have configured for the selected active evaluation.

  • Add a new tile configuration to an active evaluation.

    Select an active evaluation and choose Add to add a new tile. You can choose from one of the following tile formats:

    Numeric Data is presented in a number format.
    Comparison You select a dimension for which to show the top KPI values in comparison to each other. For example, if you select Customer as the dimension, you see a comparison of the values for the top customers contributing to this KPI.
    Trend Data is presented as a line chart illustrating the trend over time. You must enter a time dimension representing a duration (for example, month or week) to visualize this tile.
    Actual vs. Target Data is presented graphically in the form of a bullet chart that shows the current value of the KPI in relation to the target value and its thresholds.
  • Define the drill-down navigation type.

    Choose whether the drill-down for this tile leads to the SAP Smart Business generic drill-down application or another application (for example, SAP Lumira or a custom application).

    If you choose the other drill-down, you must specify the semantic object and action (and application parameters if required) with which the SAP Fiori framework can start these applications, as well as the story ID for integration with SAP Lumira (the Story ID, Semantic Object, and Action fields are only displayed if you choose Other Drill-Down).

    If the semantic object and the action are already specified for the evaluation of the tile that you are configuring, these values are carried over during tile configuration (if you have chosen Generic Drill-Down, the action is overwritten by the action for the generic drill-down).

    Note You can create and activate a tile in one step by choosing Save and Activate. Only active entities are available at runtime.
