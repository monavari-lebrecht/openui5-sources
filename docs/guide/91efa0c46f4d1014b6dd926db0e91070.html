Documentation

The following documentation is available for UI development toolkit for HTML5:

  • ABAP Server:

    To access the SAPUI5 DemoKit use this URL, replacing <host> with your host name and <port> with your port number:

    http://<host>:<port>/sap/public/bc/ui5_ui5/demokit/#docs/guide/Documentation.html

  • SAP Java Server:

    To access the UI5 DemoKit use the respective URL, replacing <host> with your host name and <port> with your port number.

  • SAP HANA Cloud:

    https://sapui5.netweaver.ondemand.com/sdk/

    Note This is the current version of the SAPUI5 DemoKit on the SAP HANA Cloud. It might not be the version of your local installation.
