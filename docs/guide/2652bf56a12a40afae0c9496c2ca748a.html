Running a SAPUI5 application

To run your SAPUI5 application, you can test it using the local runtime or you deploy it to SAP HANA Cloud Platform server.

Procedure

  1. Select your SAPUI5 application project, open the context menu and choose  Run...  Run on Server .
  2. Select either your local cloud test server or the cloud server directly
    • To test your SAPUI5 application locally, choose SAP HANA Cloud Platform local runtime
    • To deploy and run your SAPUI5 application on SAP HANA Cloud Platform, choose your server instance below hana(trial).ondemand.com and confirm with Finish.
    Your application is deployed to the respective server and started in the default browser of your Eclipse.
