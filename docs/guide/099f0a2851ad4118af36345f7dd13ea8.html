Using Extended Syntax to Add Filters and Sorters

You can use the extended syntax to add filters and sorters for aggregation binding. You can define one or multiple objects.

<table:Table rows="{
                    path:'/table', 
                    filters: {
                              path:'/table/field3', 
                              operator: 'EQ', value1: 'test'
                             }, 
                    sorter: [
                             {
                               path: '/tables/field1', 
                               descending: false
                              }, 
                              {
                               path: '/tables/field2', 
                               descending: true
                              }
                             ]
 }">
...
</table:Table>
