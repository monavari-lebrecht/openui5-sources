Create Evaluation

The Create Evaluation app allows you to create and activate evaluations for the KPIs or OPIs that you have modeled using the Create KPI app. You can only add an evaluation to an active KPI or OPI.

An evaluation is a combination of parameters, filter settings, input parameters, thresholds, targets, and trends that you apply to a KPI or an OPI. The evaluation defines what information about the KPI or OPI is visible to the SAP Smart Business user at runtime.

Key Features
  • Define the evaluation parameters.

    To save the evaluation, you must choose the KPI or OPI to which the evaluation is associated and enter an ID. To save and activate, you must also enter a title, data source, entity set, and measure.

    When you choose the KPI or OPI, the data source details (for example, the SAP HANA view, the OData service, and the entity set) are drawn from the values entered for the KPI or OPI, but you can change them here.

  • Configure the input parameters and filters.

    You can enter values for the input parameters configured in the backend. You can also set values for the dimensions to act as additional filters.

  • Configure the target, thresholds, and trend.

    You can define the target and threshold values (that is critical, warning, and target). You can enter either a fixed value or, if a measure value can be aggregated, you can enter a measure. You can only choose measures from the current entity set.

    You can either save the entity and activate it later or create and activate it at the same time by choosing Save and Activate. You can also activate and move to the tile configuration for the evaluation by choosing Activate and Add Tile.

    Note Only active evaluations are available at runtime. To visualize an evaluation, you must create a tile using the Configure KPI Tiles app.
