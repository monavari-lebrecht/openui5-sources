Rendering of Charts

The rendering of charts includes the following elements:

  • Chart title

    The chart title is taken from the longTitle property of the step object. It is displayed above any representation of a particular analysis step.

  • Titles of the x-axis and y-axis

    The general format of the title is <title (unit)>. It can be determined in one of the following ways:

    • The tile is first read from the fieldDesc property of the representation object in the analytical content configuration file, if this property has been configured.

    • If the fieldDesc property has not been configured, but the field has a corresponding sap:text annotation in the OData metadata, the system looks up the field name specified in the annotation and retrieves the sap:label value of that field. This applies to those fields for which you want to display the name rather than an ID, for example, the customer ID.

    • If the fieldDesc property has not been configured and a sap:text annotation does not exist, the system looks up the sap:label annotation of the OData metadata, which retrieves the label specified for this field in the SAP HANA model.

  • Value formatting rules:

    • All fields of type CalendarYearMonth are formatted to MM YYYY.

    • If a sap:text annotation exists in the metadata of a field, the value is formatted to textValue (keyValue), where textValue is the value of the sap:text field, and keyValue is the value of the actual field, which is given as dimension or measure.

    • In all other cases, the value is displayed without any formatting.

  • Legend

    A legend is displayed for all charts that have multiple dimensions or measures. The values can be formatted as described in the value formatting rules.

  • Labels of the x-axis and y-axis

    The rendering of the labels of the axes is defined by the behavior of the core chart. The values can be formatted as described in the value formatting rules.
