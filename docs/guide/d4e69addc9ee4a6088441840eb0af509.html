Aggregation

This control is defined by the swipeContent aggregation of the list. You can add any control as swipeContent, but keep in mind that your content cannot be higher than a list item, see the following examples:

  • Button swipeContent

    var list1 = new sap.m.List({
    swipeContent : new sap.m.Button({
        text : "Approve",
        type : "Accept",
            tap : function() { 
                /* ...approving code goes here... */

                // we are done hide the swipeContent from screen
                list1.swipeOut();
            }
        })
    }),
    ...

  • Control combination as swipeContent

    new sap.m.List({
    swipeContent : new sap.m.HBox({
        items : [
            new sap.m.Image({
                src : "images/edit.png",
                tap : ... 
            }),
            new sap.m.Image({
                src : "images/delete.png",
                tap : ...
            })
       ]
    }),
    ...
