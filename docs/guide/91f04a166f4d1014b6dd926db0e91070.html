Definining New Subclasses

For the definition of subclasses, the extend() method is available on all controls (and the base classes). To create a new control from scratch, use the following code: 

sap.ui.core.Control.extend(...)

To create a new control that inherits from Button, use the following code: 

sap.ui.commons.Button.extend(...);

The name and the definition of the new control type are the parameters of the function. The definition contains information about the control API: the properties, aggregations, and events as well as the implementation of the control methods. The definition also provides the function that creates the control's HTML structure.

Note SAPUI5 creates some methods automatically, such as getter and setter methods for properties and aggregations or methods for attaching/detaching event handlers. You can develop a non-default implementation that provides more or other functionality.
